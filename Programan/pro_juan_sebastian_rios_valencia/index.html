Script 1:

Ingresa 5 números, este script te los suma y a su vez te halla su promedio.

Script 2:

Ingresa tanta cantidad de edades que desees, este script te permitira saber cuantos son mayores de edad.

Script 3:

Ingresa tanta cantidad de numeros que deses, al finalizar te mostrara el promedio de los numeros de un digito que ingresaste.

Script 4:

Ingresa tantos numeros pares o impares que desees, este script te mostrara quien tuvo una mayor cantidad de ingresos. Recuerda que para finalizar el script se usa el número 0.

Script 5:

Este script te dice la letra equivalente al número ingresado
